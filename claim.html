Staking

Howdy! In order to unstake follow these directions:
1. Click the initialize button
2. Click the Sign into Metamask button
3. Sign the message on the Metamask popup
4. Review NFTs you have staked
5. Enter the Token ID of the token you would like to unstake
6. Select ok
You have then successfully unstaked a Cowboy!
